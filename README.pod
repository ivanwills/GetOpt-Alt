=head1 Getopt-Alt

This module provides a similar interface to Getopt::Long with some extra
extensions and syntax modernisations.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
